### README.md

**Quiz App (Memo Card)**

Простое приложение‑викторина на PyQt5: пользователь отвечает на вопросы с выбором одного варианта из четырёх. После ответа показывается результат («Правильно!»/«Неверно!»), ведётся подсчёт статистики.

#### Функциональность

* Отображение вопроса и четырёх вариантов ответа (один правильный, три неправильных).
* Случайный порядок вариантов для каждого вопроса.
* Проверка ответа и мгновенная обратная связь.
* Подсчёт:
    * общего числа вопросов (`window.total`);
    * числа правильных ответов (`window.score`);
    * рейтинга в процентах.
* Переход к следующему случайному вопросу по кнопке.

#### Структура кода

1. **Класс `Question`** — хранит вопрос и варианты ответов:
    * `question` — текст вопроса;
    * `right_answer` — правильный ответ;
    * `wrong1`, `wrong2`, `wrong3` — неправильные ответы.
2. **Список `questions_list`** — коллекция объектов `Question` с тестовыми данными.
3. **Виджеты PyQt5** — интерфейс:
    * `QLabel` — для текста вопроса и результатов;
    * `QRadioButton` (в `QButtonGroup`) — варианты ответов;
    * `QPushButton` (`btn_OK`) — кнопка «Ответить»/«Следующий вопрос»;
    * группы `QGroupBox` — логическая группировка элементов (панель вопросов и панель результатов).
4. **Ключевые функции**:
    * `ask(q: Question)` — заполняет интерфейс данными вопроса, перемешивает варианты;
    * `show_question()` / `show_result()` — переключает видимость панелей (вопрос/результат);
    * `check_answer()` — проверяет выбор пользователя, обновляет счёт, выводит результат;
    * `next_question()` — выбирает случайный вопрос из списка, запускает показ;
    * `click_OK()` — обработчик кнопки: запускает проверку или следующий вопрос.
5. **Главное окно (`window`)** — собирает все элементы, устанавливает заголовок и размер, запускает первый вопрос.

#### Использование

1. Убедитесь, что установлен **Python** и библиотека **PyQt5**:
   ```bash
   pip install PyQt5
   ```
2. Сохраните код в файл (например, `memo_card.py`).
3. Запустите:
   ```bash
   python memo_card.py
   ```
4. В окне приложения:
    * Прочитайте вопрос.
    * Выберите один вариант ответа.
    * Нажмите кнопку **«Ответить»**.
    * Ознакомьтесь с результатом («Правильно!» или «Неверно!»).
    * Нажмите **«Следующий вопрос»** для продолжения.

#### Примечания

* Вопросы хранятся в коде (список `questions_list`). Для расширения добавьте новые экземпляры `Question`.
* Статистика (счёт и рейтинг) выводится в консоль.
* Текущая логика допускает повтор вопросов. Для исключения повторов можно модифицировать `next_question()`, отслеживая уже заданные вопросы.
